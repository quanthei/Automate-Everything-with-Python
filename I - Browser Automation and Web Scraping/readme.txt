II - How to run the script?

    1. Select an option

        Option 1 : run it online on replit
            --> https://replit.com/

        Option 2 : run on a local IDE :
            1. Find my browser version --> https://www.whatismybrowser.com
            2. Dl the corresponding webdriver version --> https://getwebdriver.com
            3. Put the .exe in the dl folder (for example)
            4. Adapt the code (i.e. script in 'scripts' folder)
    
    2. Install dependancies
    
    In the bash terminal :
        pip install pytest
        pip install selenium
